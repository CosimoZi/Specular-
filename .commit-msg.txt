replace Pando with custom AST; Shenhe panel ATK = 2,717 correct

After hitting one too many Pando edge cases (the dst:null cond
propagation, sheet routing quirks, +100% ATK stubs we had to strip),
shifted to a from-scratch expression engine in src/calc/. Vendored GO
stays in tree for now as a reference for non-Shenhe characters.

Architecture:
  src/calc/ast.ts      12-op discriminated union + custom escape hatch.
                       Fail-fast: missing var throws, div0 throws.
                       evaluate, simplify (with constant folding),
                       pretty, freeVars all included.
  src/calc/scope.ts    Plain Map<string, number> with parent chain
                       and add/setMax accumulators for buffs.
  src/calc/build.ts    11-phase pipeline: char + weapon + artifact
                       data fill -> AST eval for base/premod/final.
  src/calc/formula.ts  Damage formula evaluator: base x dmgBonus x
                       critMulti x defMulti x resMulti.
  src/calc/sheets/     Per-character / weapon / set buff sheets.
                       NoblesseOblige, BlizzardStrayer,
                       TenacityOfTheMillelith, CalamityQueller,
                       Shenhe (+ damage formulas).
  src/calc/team-adapter.ts  Shapes BuildResult into the existing
                            GoComputeResult interface so /team UI
                            renders unchanged for characters with
                            src/calc/ sheets (Shenhe today); falls
                            through to legacy go-calc for others.

End-to-end verification with Shenhe L90 A6 C0 + Calamity Queller

1 CD, 1 ATK%, 1 ATK flat at max-tier rolls):

  premod.atk_ = 1.211     (28.8% char A6 + 16.5% CQ substat
                           + 46.6% sands main + 29.15% subs)
  premod.atk.flat = 408.25 (311 plume + 5x19.45 subs)
  final.atk   = 2717.16   matches in-game expected
  HP          = 17,773
  DEF         = 830
  CR          = 55.5%
  CD          = 88.9%
  Cryo DMG    = 58.6%

Damage formulas with no conds (no artifacts) match legacy GO values
exactly:
  normal_0 = 583, normal_4 = 885, charged = 1492,
  plunging_high = 2153, skill_press = 1914, skill_hold = 2596,

  - A1 burst field: cryo skill_press up, physical normal_0 up (RES shred)
  - C4 stacks (C6): own skill DMG up, burst/normals unchanged
  - CQ stack=6 isActive=on/off: ATK% +19.2% / +38.4% as expected
  - 4pc BlizzardStrayer enemyCryo/enemyFrozen: +20% / +40% CR
  - NO 2pc / 4pc set4: burst DMG / team ATK%

Test count: 70/71 pass. The one failure is a legacy GO test
(shenhe-equipment.test.ts NoblesseOblige team-buff propagation)
